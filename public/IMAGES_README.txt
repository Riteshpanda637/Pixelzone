IMAGE OPTIMIZATION TIPS:
------------------------
- Use JPG format for photos (better compression)
- Use WebP format for even better compression
- Keep file size under 500KB per image
- Optimize using tools like:
  * TinyPNG (https://tinypng.com/)
  * Squoosh (https://squoosh.app/)
  * ImageOptim (Mac)
